## What is this?

This package exposes states to parse whitespace.

## When should I use this?

This package may be useful when making your own docmark extensions.

## Install

This package is [ESM only][esm].

In Node.js (version 20+) with [yarn][]:

```sh
yarn add @flex-development/docmark-factory-whitespace
```

<blockquote>
  <small>
    See <a href='https://yarnpkg.com/protocol/git'>Git - Protocols | Yarn</a>
    &nbsp;for details regarding installing from Git.
  </small>
</blockquote>

In Deno with [`esm.sh`][esmsh]:

```ts
import { factoryWhitespace } from 'https://esm.sh/@flex-development/docmark-factory-whitespace'
```

In browsers with [`esm.sh`][esmsh]:

```html
<script type="module">
  import { factoryWhitespace } from 'https://esm.sh/@flex-development/docmark-factory-whitespace'
</script>
```
